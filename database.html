Introduction to Databases (ISCG5423)

Level: 5
Credits: 15 (0.125 EFTS)
Compulsory: Yes
Aim: To provide students with an introduction to database systems.
Pre-requisites / Co-requisites:
Restriction: ISCG5236.
Topics include: Data, data types, set theory, DBMS terms, entity relationship diagrams, creation of tables and queries using SQL DDL & SQL DML Other topics as negotiated
